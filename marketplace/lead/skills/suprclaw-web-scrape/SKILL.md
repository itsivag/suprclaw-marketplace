# SuprClaw Web Scrape

Use `web_scrape` to extract clean markdown from any public URL.

- Fast, read-only. No browser session required.
- Use for: research, reading articles, SEO analysis, data extraction.
- Do NOT use for: sites requiring login, JavaScript-heavy SPAs (use `cloud_browser_open` instead).
